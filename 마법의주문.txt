2. 상체/하체 분리 (록맨의 정석)
Base Layer: Idle, Run, Jump만 냅둡니다. (사격 모션 다 삭제!)

Shooting Layer:

Empty State (이름: None)를 하나 만들고 기본 상태(주황색)로 둡니다.

Shoot, Shoot_Run 등을 여기 배치합니다.

None ↔ Shoot_Run 화살표를 만들고 isShooting 파라미터로 연결합니다.

💡 왜 이렇게 하면 해결되나요?
이렇게 분리하면 유니티는 두 개의 애니메이션을 동시에 돌립니다.

하체(Base): 총알이 사라지든 말든 Run 애니메이션을 0.1초도 멈추지 않고 계속 재생합니다.

상체(Shooting): isShooting이 켜진 동안만 달리기 모션 위에 사격 포즈를 덧씌웁니다.

결국 총알이 빨리 사라져서 isShooting이 아주 미세하게 흔들리더라도, 다리는 Base Layer 소속이라 절대로 프레임이 튀지 않게 됩니다.
